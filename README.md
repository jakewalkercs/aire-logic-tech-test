## Prerequisites

* Python 3 <br />
* Clone this repo <br />

## Getting Started

* Install dependencies: <br />
  $ pip install pipenv <br />
  $ pipenv install <br />

* Activate virtual environment: <br />
  $ pipenv shell <br />

* Start the app: <br />
  $ python src/main.py <br />
